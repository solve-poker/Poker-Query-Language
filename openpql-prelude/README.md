# openpql-prelude

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Rust](https://img.shields.io/badge/rust-1.85+-blue.svg)](https://www.rust-lang.org)

Core poker types — cards, hands, boards, and evaluators for Hold'em, Short Deck, and Omaha.

Bit-packed card representation and fast hand evaluation, used by the rest of the Open PQL workspace.

### Optional features

- `rand` (default) — RNG for Monte Carlo simulations
- `quickcheck` — property-testing support
